## 정답 및 해설

1. **정답: 2 (`#`)** - 파이썬은 `#` 기호로 주석을 시작합니다.
2. **정답: 3 (Boolean)** - `True` 또는 `False` 값을 가지는 자료형은 불리언입니다.
3. **정답: 2 (문자열)** - `input()` 함수는 사용자가 숫자를 입력해도 무조건 문자열(String)로 반환합니다. 계산하려면 `int()`로 변환해야 합니다.
4. **정답: 4 (`append()`)** - 리스트의 맨 끝에 요소를 추가할 때는 `append()`를 사용합니다.
5. **정답: 3 (3번)** - 리스트의 항목 수(3개)만큼 반복되므로 출력 자체는 3번 일어납니다. ("오렌지" 포함 전체 출력 횟수 중 "사과"라고 묻는다면 2번이겠지만, 반복문 자체의 실행 횟수나 전체 출력을 묻는 문맥이라면 3번 루프가 돕니다. 문제 의도상 전체 반복 횟수를 의미하거나, 단순히 코드 실행 결과 화면에는 "사과", "사과", "오렌지"가 찍힙니다.)
   * *수정 질문 의도*: 만약 "사과"라는 글자가 몇 번 찍히냐는 질문이라면 2번입니다.
6. **정답: 2 ("B")** - 85는 90보다 작고 80보다 크거나 같으므로 `elif score >= 80:` 조건에 걸립니다.
7. **정답: 1 (`return`)** - 값을 반환할 때는 `return` 키워드를 사용합니다.
8. **정답: 3 (`__init__`)** - 파이썬 클래스의 생성자(초기화 메서드)는 `__init__`입니다.
9. **정답: 3 (`my_dog.play()`)** - 예제 코드에서 `play()` 메서드는 `self.energy -= 10`을 수행합니다.
10. **정답: 2 (`except:`)** - 파이썬에서는 `try-except` 구문을 사용하여 예외를 처리합니다.
